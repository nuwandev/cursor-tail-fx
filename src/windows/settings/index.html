Project Details

CursorTrail draws inspiration from beautiful modern web interfaces and brings native, optimized cursor trails directly to your operating system via transparent layered windows.

By leveraging WebGL Instanced Arrays, memory footprint is kept exceptionally low while achieving silky smooth 144hz rendering across all graphical effects.

Designed and built for peak aesthetics.
