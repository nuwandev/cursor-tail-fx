How it works

Cursora renders all effects using WebGL 2 instanced drawing — a single GPU draw call is designed to render up to 1,000 particles at a 60 fps target on typical hardware, while keeping CPU overhead low.

The Rust backend uses adaptive polling — tracking the cursor at 120 Hz when active, dropping to 10 Hz during idle — so CPU usage when you're not moving the mouse stays very low.

New effects can be added as a single .ts file and automatically appear in the settings with no changes required to existing code.
